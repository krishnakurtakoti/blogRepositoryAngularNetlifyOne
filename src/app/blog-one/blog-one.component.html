Deploying Angular app from Github on Netlify

08/08/2019

My Account:

This writing is a reference for me on deploying Angular app from Github on Netlify.I have successfully deployed the Angular app on Netlify for free.

Step 1:

Prerequisites :

1. Git installed

2. Node.js installed

3. Visual Studio Code

4. Angular installed

5. Github Account

6. Netlify Account

You can use Github as OAuth provider,i.e,if you have Github account,you can directly Register and Login into Netlify.

Starting from scratch

If you do not have Git on Windows?

Following this video "Install Git, Clone a project (Windows)" https://www.youtube.com/watch?v=Jge57UHvYto&list=WL&index=3 for Git Installation on Windows10 worked for me.

Only one necessary change to make if git is not recognized in Windows is :

Choose "Use Git and optional Unix tools from the Windows Command Prompt" as shown below

After the Successful installation,Right click Windows10 start option and click search.Type environment and you will see Edit the system environment variables (Control Panel)and click Environment Variables.

You should see the installed Git in the path as shown below:

OK.Now test the git by typing git in new command prompt(cmd):

All set.

One more task

In the Git Bash,Configuration of your Github user name and email must be done.

Follow the below video "GIT: Installation, Configuration and First Commit" https://www.youtube.com/watch?v=QqP7YZlZEOo to configure your Github username and email with the Git.

First config username and email to connect with the github.

git config --global user.name "username"

git config --global user.email "email"

git git add .

git commit -m 'initial commit'

Create a new repository in Github without a readme.

git remote add origin https://github.com/username/repositoryName.git

git push -u origin master

git log to see the commits.

All set for a Github Push.

If you do not have Node.js on Windows?

Follow the tutorial on "How to Install and Use Node.js and npm (Mac, Windows, Linux)" in https://www.taniarascia.com/how-to-install-and-use-node-js-and-npm-mac-and-windows/.

It says "Installing everything on Windows is a breeze.Install Node.js and npm Node.js and npm can be installed from a download link. Go to the Node installation page, and download the Node installer. I have a 64-bit Windows 10 OS, so I chose that one."

"Once it's done, you can test to see both node and npm functioning by opening PowerShell (or any shell) and typing node -v andnpm -v, which will check the version number."

All set.

If you do not have Visual Studio Code on Windows?

Vist the link https://code.visualstudio.com/

Click Download for Windows and run the application.

All set.

If you do not have Angular?

Angular installation is managed by Node.js(npm).So,with Node.js installed,you can get Angular installed with the following code:

npm install -g @angular/cli

Make a new folder AngularTest.

Open the folder in Visual Studio Code.

Open a new terminal in Visual Studio Code.

For Angular CLI installation type, npm install -g @angular/cli in the terminal.

To create a new Angular project,type ng new helloWorld in the terminal.

On Successful installation,you can see

After creation,compile the app with typing cd helloWorld in the terminal.

See the browser localhost:4200

During Angular CLI installation,I encountered the below error.

https://stackoverflow.com/questions/47675478/npm-install-errorunexpected-end-of-json-input-while-parsing-near-nt-webpack

When Creating a new Angular 5 project: node version: 8.9.2, npm version: 5.5.1, My Command is npm install -g @angular/cli the error is

npm ERR! Unexpected end of JSON input while parsing near '...nt-webpack-plugin":"0'

Answers

npm cache clean --force

then run

npm install -g @angular/cli@latest

All set.

If you do not have Github Account?

Go and sign up for new Github Account.

All set.

If you do not have Netlify Account?

Go and sign up for new Netlify Account using your Github Account for free.

All set.
